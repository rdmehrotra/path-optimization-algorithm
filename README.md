# Constrained Path Optimization Algorithm (C & RISC-V)

This project implements a minimum-cost path algorithm on a constrained grid.

## Features
- Dynamic programming approach
- Optimized for low instruction count (RISC-V)
- Memory-efficient implementation
- Tested across multiple grid configurations

## Tech Stack
- C
- RISC-V Assembly

## Description
The algorithm computes the optimal path from the center of a constrained diamond grid to its edge, minimizing total cost while respecting movement constraints.
